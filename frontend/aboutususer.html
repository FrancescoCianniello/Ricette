Welcome to La Ricetta per te!

The "La Ricetta per te" app is a new online platform for those who want to experiment with recipes of all kinds or for those who need help cooking something quick.

How can the user use the app?

Users, after registering and entering their private area, can access a large collection of recipes.The user can decide whether to search for the recipe through the filters that customize the search or can proceed to a search by name or by scrolling through the list. At this point, if the recipe is needed at that moment, the user can also use only the view, while if he has to use it in the future, there is the option to select the recipe and place it in the list of saved recipes. Each recipe is associated with the chef who published it, and the user can take advantage of a messaging system to be able to ask the chef for suggestions or clarifications. The user can also decide to view only the chef's profile or to follow him, i.e. to receive updates on the recipes he will publish in the future. A final option available to the user is to be able to create their own shopping list, adding or removing the ingredients in the database.

How can the chef use the app?

The chef also has his own authentication system and is allowed to publish recipes or remove them if they receive low ratings.

Additional information

The app is completely free and simple to use for any category of user.
